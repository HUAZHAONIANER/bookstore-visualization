# 大型图书城数据可视化系统 - 结课项目

## 1. 项目概述

### 1.1 项目背景
本项目是基于《Python数据可视化》课程内容设计的结课项目，以大型图书城为业务背景，开发一套完整的数据可视化系统。

### 1.2 项目目标
- 构建包含前台用户界面和后台管理界面的完整系统
- 实现多种数据可视化图表，展示图书城运营数据
- 提供交互功能，支持数据筛选、动态更新和悬停提示

### 1.3 技术栈
- **框架**: Streamlit 1.32.0
- **数据处理**: Pandas 2.2.1, NumPy 1.26.4
- **可视化**: Plotly 5.18.0, Matplotlib 3.8.3, Seaborn 0.13.2
- **运行环境**: Python 3.8+

## 2. 系统架构

### 2.1 架构设计
```
├── app.py                    # 主应用入口
├── config/                   # 配置文件目录
│   └── config.py             # 系统配置
├── docs/                     # 文档目录
│   └── PROJECT_DOCUMENT.md   # 项目文档
```

### 2.2 模块职责

| 模块 | 职责 | 文件位置 |
|------|------|----------|
| 数据层 | 模拟数据生成 | `src/data/mock_data.py` |
| 处理层 | 数据清洗、转换、分析 | `src/modules/data_processor.py` |
| 可视化层 | 图表创建与渲染 | `src/modules/visualization.py` |
| 界面层 | 用户交互界面 | `src/ui/frontend.py`, `src/ui/backend.py` |

## 3. 功能模块

### 3.1 前台用户界面
面向普通用户的数据展示与交互平台

#### 3.1.1 功能特性
- **数据概览卡片**: 展示关键运营指标（销售额、订单数、满意度、周转率）
- **销售趋势分析**: 月度销售趋势折线图，支持类别筛选
- **品类分析**: 品类销售占比饼图、各类别销售额对比柱状图
- **客户分析**: 一周客流量变化、会员到店情况、日均客单价分析
- **畅销排行**: Top N畅销书籍排行，支持类别筛选和评分关系分析

### 3.2 后台管理界面
面向管理员的数据管理与可视化配置平台

#### 3.2.1 功能特性
- **数据概览**: 关键指标相关性热力图、库存状态分布
- **库存管理**: 库存数据表格展示，支持区域筛选
- **员工管理**: 部门人员分布、薪资总额分析
- **数据关联分析**: 销售数量与销售额关系、库存利用率热力图
- **可视化配置**: 图表配色、刷新间隔、显示设置等配置项
- **系统状态监控**: 健康度、数据完整性、服务可用性仪表盘

## 4. 可视化功能

### 4.1 图表类型

| 图表类型 | 应用场景 | 实现位置 |
|----------|----------|----------|
| 折线图 | 销售趋势、客流量变化 | `visualization.py:create_line_chart` |
| 柱状图 | 品类对比、库存分布、薪资分析 | `visualization.py:create_bar_chart` |
| 饼图 | 销售占比、库存状态分布 | `visualization.py:create_pie_chart` |
| 散点图 | 价格销量关系、库存周转关系 | `visualization.py:create_scatter_chart` |
| 热力图 | 指标相关性、库存利用率 | `visualization.py:create_heatmap` |
| 仪表盘 | 系统状态监控 | `visualization.py:create_gauge_chart` |

### 4.2 交互功能

| 交互类型 | 实现方式 | 应用位置 |
|----------|----------|----------|
| 数据筛选 | Streamlit选择框、滑块 | 前台各Tab页 |
| 动态更新 | 选择框触发重新渲染 | 前台各Tab页 |
| 悬停提示 | Plotly原生hover功能 | 所有图表 |
| 图例交互 | Plotly图例点击筛选 | 所有多类别图表 |
| 缩放平移 | Plotly工具栏 | 所有图表 |

## 6. 部署与运行

### 6.1 环境要求
- Python 3.8 及以上版本
- 建议使用虚拟环境

### 6.2 安装步骤

```bash
# 进入项目目录
cd bookstore_visualization

# 创建虚拟环境（可选）
python -m venv venv

# 激活虚拟环境
# Windows: venv\Scripts\activate
# Linux/Mac: source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 6.3 运行方式

```bash
# 启动Streamlit应用
streamlit run app.py
```

运行成功后，打开浏览器访问显示的本地地址（通常是 http://localhost:8501）

### 6.4 界面切换

在左侧侧边栏选择：
- **前台用户界面**: 面向普通用户的数据展示
- **后台管理界面**: 面向管理员的数据管理

## 7. 代码规范

### 7.1 命名规范
- **文件命名**: 小写字母+下划线，如 `data_processor.py`
- **类名**: 大驼峰式，如 `DataProcessor`
- **函数名**: 小写字母+下划线，如 `get_monthly_sales`
- **变量名**: 小写字母+下划线，如 `sales_data`

### 7.2 代码结构
- 每个函数不超过50行
- 类方法按功能分组
- 适当添加空行分隔逻辑块

### 7.3 注释规范
- 模块级注释：说明模块功能和使用方法
- 类注释：说明类的职责和主要方法
- 函数注释：说明参数、返回值和功能

## 8. 项目亮点

### 8.1 技术亮点
1. **模块化设计**: 清晰的分层架构，代码易于维护和扩展
2. **交互式可视化**: 丰富的交互功能，提升用户体验
3. **响应式布局**: 使用Streamlit实现自适应界面
4. **数据模拟**: 完整的模拟数据生成，无需数据库支持

### 8.2 功能亮点
1. **多维度分析**: 销售、客户、库存、员工多维度数据展示
2. **实时监控**: 系统状态仪表盘监控
3. **配置灵活**: 支持可视化配置自定义
4. **专业图表**: 符合企业级应用标准的可视化效果
